# landmark-detection---version-1
Real-Time Pose Landmark Detection for Motion-Based Combat Interaction
Built using Python, OpenCV, and MediaPipe Pose Estimation, the system detects body movements in real time and maps them to game-style actions such as:

• Shield (Squat Detection)
• Jump Detection
• Dodge Left / Dodge Right
• Bow Attack Detection (supports both left-handed and right-handed users)

The project uses pose landmarks from a webcam feed to recognize actions and display them in real time.

This project helped me gain hands-on experience with:

* Computer Vision
* Pose Estimation
* Real-Time Action Recognition
* OpenCV and MediaPipe
